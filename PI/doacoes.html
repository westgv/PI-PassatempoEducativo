DOAÇÕES

Como doar

Você pode ajudar a Passatempo Educativo através de doações!

Todos os valores doados são aplicados integralmente nos nossos programas de atendimento.

Seu apoio será transformado em:

  • Compra de novos livros para reposição de exemplares do Projeto Livros em Ação
  • Material de consumo para atividades multidisciplinares.
  • Confecção de novos jogos do Projeto Jogoteca
  • Compra de materiais esportivos para Projeto Futebol Ciência
  • Despesas com hospedagem de sites
  • Transporte para projetos fora da cidade de São Paulo
  • Desenvolvimento de novos projetos educativos
  • Materiais para o Projeto Reforço Interativo
